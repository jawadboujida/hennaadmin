⚙️ Eenmalige setup — 5 minuten

1
Ga naar sheets.google.com en maak een nieuw spreadsheet. Geef de eerste rij headers: Naam Aanwezig Henna Datum
2
Klik in het spreadsheet op Extensies → Apps Script. Verwijder alle code en plak de script code die je van Jawad hebt gekregen. Sla op.
3
Klik op Implementeren → Nieuwe implementatie. Type: Webapp. Toegang: Iedereen. Kopieer de URL.
4
Plak de URL hierboven in het veld Google Sheet URL en klik op Vernieuwen.
